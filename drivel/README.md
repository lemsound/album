# 三文抒情 — Web アルバムプレイヤー

黒背景・月イエローの差し色で、ブラウザ上でアルバムを再生するアプリです。
PC／スマホ両対応。歌詞表示・シーク・シャッフル・ループ（アルバム／単曲）・共有リンク対応。

公開URL（GitHub Pages）: https://lemsound.github.io/album/drivel/

------------------------------------------------------------
## 1. ファイルの置き場所
`lemsound/album` リポジトリの `drivel/` フォルダ（音源・歌詞・jacket がある場所）に、
この 8 ファイルを **そのまま全部** 上書きで入れてください。

    album/drivel/
    ├─ index.html        ← アプリ本体（ここを開くと再生画面）
    ├─ style.css
    ├─ app.js
    ├─ sw.js             ← キャッシュ管理（通信量節約）
    ├─ manifest.json     ← 曲リスト等（build.py が自動生成）
    ├─ design.txt        ← ★色・フォント・タイトルサイズを変えるファイル
    ├─ timing.txt        ← ★曲の頭/尻を秒で微調整するファイル
    ├─ build.py          ← 曲を足したとき等に走らせるスクリプト
    │
    ├─ 三文抒情.album     （既存）
    ├─ jacket.jpeg        （既存）
    ├─ 曲順.txt           （既存）
    ├─ 天井.mp3 / 天井.txt （既存）… ほか各曲

※ README.md はGitHubでの説明用。アプリの動作には使いません。

------------------------------------------------------------
## 2. デザインを変える → design.txt
「=」の右側だけ書き換えて保存 → 再読み込み。

    背景色   = #000000
    サブ色   = #161616
    差し色   = #e8c45a   （月イエロー）
    文字色   = #f2f2f2
    淡色     = #8c8c8c
    アルバム名 = 明朝       （明朝 / ゴシック / セリフ / サンセリフ）
    歌詞       = ゴシック
    アルバム名サイズ = 4.4   （数字だけでrem。3=控えめ / 4.4=標準 / 6=特大）

・「アルバム名サイズ」はタイトル文字の大きさ。スマホでは自動で画面内に収まります。
・曲ごとに歌詞フォントを変えたいときは下に追記:  歌詞@天井 = ゴシック

------------------------------------------------------------
## 3. 曲の頭/尻を調整 → timing.txt
プラス(+)で無音を足す、マイナス(-)でカット。単位は秒。

    天井    頭=+0.0  尻=+0.3   → 曲尻に 0.3 秒の無音
    灰色    頭=-0.5  尻=+0.0   → 曲頭を 0.5 秒カット

------------------------------------------------------------
## 4. 曲を追加・差し替えたら → build.py
音源／歌詞／曲順.txt を変えたら、このスクリプトを 1 回走らせて
manifest.json を更新します（再生時間の再計算もこれが行います）。

    pip install mutagen        # 初回のみ（mp3 の長さ取得に使用）
    python build.py            # album/drivel/ 内で実行

  ・design.txt / timing.txt は **既にあれば上書きしません**（安心）。
  ・新しい曲の行は timing.txt に追記されます。

------------------------------------------------------------
## 5. ローカルで確認するとき
`index.html` を直接ダブルクリック（file://）では動きません。
簡易サーバ経由で開いてください:

    cd album/drivel
    python -m http.server 8000
    → ブラウザで http://localhost:8000/

※ ローカルの簡易サーバはシーク（途中ジャンプ）に対応していない場合があります。
   GitHub Pages 上では正常にシークできます。

------------------------------------------------------------
## 6. 操作のメモ
・上部の丸い再生ボタン … アルバムを頭から再生／一時停止（再生中はアイコンが変化）。
・「歌詞」ボタン（シャッフル等の並び）… 今の曲の歌詞を開く／閉じる。
・シークバー … クリックでもドラッグでも移動可。スマホは指に追従。
・音量 … 下部プレイヤー右（PC）。スマホは端末側の音量ボタンを使用。
・共有ボタン … いま開いている曲付きの URL（?t=曲名）をコピー。SNSでジャケットがサムネに出ます。
・リロードすると、最初にアルバムを開いた画面に戻ります（曲データのキャッシュは保持）。
